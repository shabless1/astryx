## HOW TO USE THIS FILE

Each track has two fields you paste directly into Suno:
- **STYLE** → paste into Suno's "Style of Music" field
- **AVOID** → paste into Suno's "Style to Avoid" field
- Set lyrics to: `[Instrumental]`
- Target duration: **3:30–4:00** per track (use Suno's Extend feature to reach 4 min if needed)

**After generating:** Save each file as `SUN_[STATE]_[##].mp3` in `public/audio/library/sun/`

**BPM derivation:** 126.22 Hz ÷ 2 = 63.1 BPM. The tempo IS the frequency octaved down — this keeps every Sun track mathematically anchored to the Cousto Hz. Corrective tracks use the regulator planet's derived BPM.

**Key for each state:**
- Natural: B Lydian (radiant, uplifted, golden)
- Excess corrective: G# / Ab Phrygian (Moon character — cooling, floating)
- Deficiency corrective: D Aeolian/Dorian (Mars character — warming, activating)
- Blocked corrective: F# Ionian / A Mixolydian (Jupiter+Venus — expansive, opening)
